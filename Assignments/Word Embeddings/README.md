# Word Embeddings


### Contents
1. __*glove*__ - module for GloVe word vectorization (NumPy implementation)
2. __*word2vec*__ - module for Word2Vec vectorization with Keras
3. __*model_comparisons*__ - module with algorithms for evaluating embeddings 
4. __*Word_Embeddings_Baseline.ipynb*__ - RNN for IMDB dataset
5. __*Word_Embeddings_Regularised.ipynb*__ - RNN for IMDB dataset with regularization for reducing overfitting
6. __*Word_Embeddings_Glove.ipynb*__ - Notebook for testing the *glove*  module
7. __*Word_Embeddings_word2vec.ipynb*__ - Notebook for training embeddings using word2vec and using them for an RNN for the IMDB dataset.
